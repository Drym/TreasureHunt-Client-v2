# TreasureHunt-Client

## Comment installer et exécuter le client

### Prérequis
- git
- npm
- node
- bower

### Lancement
- Dans un terminal, installez les dépendances en lançant la commande :
`npm install`
- Puis, lancez le serveur web avec la commande :
`npm start`
- Ouvrez une brower et rendez-vous à l’adresse : 
`http://localhost:3000/`
